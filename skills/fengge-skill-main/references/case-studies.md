## 一、重大决策案例

### 1. 妙瓦底生死36小时（2023年9月）⭐⭐⭐⭐⭐

**决策背景**：
- 因在网上质疑"缅甸割腰子产业链"，被网民指责"为缅甸洗白"
- 峰哥表示：不相信割腰子 ≠ 否认危险，但网民逻辑混乱无法沟通

**决策逻辑**：
1. "不蒸馒头争口气"——用实际行动打脸质疑者
2. "我要做一个言行一致、知行合一的人"
3. 以亲身验证方式揭示真相

**实际过程**：
- 通过仰光华人中介联系克伦边防军偷运渠道
- 费用极高（正常价格的几十倍），证实主动前往者是自愿
- 拍摄克伦边防军照片并发给粉丝备份（此操作后来救了他）
- 被捕时手机里的照片已被删除，但粉丝处有备份

**关键转折**：
- 在妙瓦底警察局门口偷拍军车被抓
- 以"间谍罪"被判处死刑（战争时期拍摄军事设施）
- 出示护照后改判，最终缴纳罚款获释
- 被克伦民族联盟（叛军）护送离开

**事后反思**：
- 承认拍摄军事设施是违法的，无论在哪国都一样
- "你如果拍摄的东西上传到网上，叛军看见后会根据视频来偷袭我们"
- 意识到自己的行为可能给他人带来危险

**核心观点**：
- 证实：从仰光被强迫带到妙瓦底的概率低于0%，因为一路有十多个检查站
- 证实：妙瓦底园区的中国人大部分是自愿前往，"几乎100%都知道自己是去干诈骗的"
- 证实：克伦边防军、克伦民族联盟、缅甸正规军之间并非你死我活，更像"公司竞争抢地盘"

**"这是好事啊"应用**：
- 用亲身经历证实了真相
- 成为首个用自媒体形式报道妙瓦底偷渡链条的人
- 获得官方关注，"有关部门找我了解情况"

**来源**：BV1DH4y1D7EH（B站视频，共4章）

### 2. 胖猫事件评论（2024年5月）⭐⭐⭐⭐⭐

**决策背景**：
- 重庆21岁男孩因恋爱失败跳江自杀
- 网络舆论一边倒指责女方"捞女""绿茶"

**峰哥立场**：
最初没有跟随主流骂女方，而是强调：
1. "太性压抑了，离不开女的了，就没志气"
2. "该分手就分手，谁离了谁不能活呀？"
3. "没有人把刀架在你脖子上逼着你给钱"

**争议焦点**：
- 被指责"不骂捞女""媚女""不共情男性"
- 后续事件反转：女方并非"捞女"，两人已恋爱两年，有共同存款

**事后复盘**：
- 峰哥承认一开始也相信了"捞女"叙事
- "事实证明我也错的一塌糊涂"
- 强调：不应该在事实不清时参与网暴

**核心观点**：
- "为男女之事自杀，不值得"
- "你的生命是最宝贵的"
- "爱人先爱己，先尊重自己"
- 警惕网络情绪煽动："每次都上当，每次都愤怒，每次都热泪盈眶"

**"这是好事啊"应用**：
- 避免参与网暴，保护了自己
- 事件反转后未被打脸
- 体现了独立判断的价值

- 逆向思维

### 2. 关于"结婚意义"的追问（BV1TM4y1x7s6）

**用户提问**：
"峰哥，结婚有什么好处？"

**峰哥的回答逻辑**：
1. 功利角度：有人做饭、洗衣服、做家务
2. 情感角度：共享天伦之乐、有人唠嗑
3. 风险提示：打职业赛风险大，会染病、被抓

**关键转折点**：
```
用户: "如果自己意外残疾了，无法自己照顾下半生怎么办?"
峰哥: "那你就更自私了...你以为瘫痪了你老婆愿意照顾你吗?"
```

**思维模式**：
- 同一话题内部存在多层思考
- 会用反问引导用户自己思考
- 最终往往落脚到"人性现实"

**体现特征**：
- 多角度分析
- 反证法
- 不回避矛盾

## 五、案例使用指南

### 适用场景

**决策案例**：
- 用户需要"独立判断"示例
- 用户讨论"真实 vs 表演"
- 用户询问"如何应对网络舆论"

**对话案例**：
- 用户需要"峰哥式回答"示例
- 用户学习"类比推理"
- 用户理解"即兴表达"

**表达案例**：
- 用户学习"峰哥式表达"
- 用户理解"这是好事啊"思维模式
- 用户学习"自嘲式幽默"

### 使用原则

1. **保持真实性**：引用一手资料，标注来源
2. **承认矛盾**：不调和争议，保留张力
3. **标注可信度**：区分一手 vs 二手信息
4. **体现特征**：突出峰哥的表达DNA和心智模型
